## 1. Нейрофизиологические основания ограничений по времени

Произвольное внимание дошкольника — ограниченный ресурс. Это связано с незрелостью префронтальной коры (зона произвольного контроля) и с высокой скоростью истощения тормозных нейронных механизмов.

### Ключевые данные

**Дошкольный возраст (5–7 лет):**
- Максимальное непрерывное произвольное внимание на интеллектуальной задаче: 5–10 минут
- После 10 минут без смены деятельности — нарастающая рассеянность
- Пиковая работоспособность: 9:00–12:00, 16:00–18:00
- Наихудшее состояние: сразу после пробуждения, сразу после еды, вечер после 19:00

**Младший школьный возраст (7–8 лет):**
- Произвольное внимание: 10–15 минут
- После 15 минут — необходима смена деятельности или 1–2 минуты «разгрузки»
- Лучший результат достигается при чередовании типов активности каждые 3–5 минут

### Специфика речевой работы

Речевые задания — одни из наиболее когнитивно нагрузочных для детей с речевыми нарушениями. Ребёнок с дислалией тратит на произношение звука значительно больше когнитивного ресурса, чем нормотипичный ребёнок, — потому что навык ещё не автоматизирован и требует сознательного контроля.

**Практический вывод:** нагрузка, которая кажется «лёгкой» снаружи (сказать 10 слов с Р), для ребёнка с ротацизмом — как для взрослого 30 минут концентрированного иностранного языка.

## 2. Нормативы длительности по возрасту

### 5 лет (4 года 6 месяцев — 5 лет 11 месяцев)

| Параметр | Значение |
|----------|----------|
| Длительность сессии | 7–9 минут |
| Количество активностей | 2–3 |
| Максимум произносительных попыток | 4–6 |
| Смена активности | каждые 2–2,5 минуты |
| Длина инструкции | 1–2 слова / иконка |
| Пауза внутри сессии | 1 мини-пауза (20–30 сек) |
| Финальное задание | обязательно лёгкое |

**Допустимые шаблоны:** ART, AR, BR (лёгкие), L&C уровень 1, RAM уровень 1, RHY.
**Недопустимые:** MP, NQ, SC уровень 3, RAM уровень 3, фонемный анализ.

### 6 лет (5 лет 6 месяцев — 6 лет 11 месяцев)

| Параметр | Значение |
|----------|----------|
| Длительность сессии | 10–12 минут |
| Количество активностей | 3–4 |
| Максимум произносительных попыток | 6–8 |
| Смена активности | каждые 2,5–3 минуты |
| Пауза внутри сессии | 1–2 мини-паузы |

**Допустимые шаблоны:** все до уровня 2. Добавляются MEM, BNG, D&M.

### 7 лет (6 лет 6 месяцев — 7 лет 11 месяцев)

| Параметр | Значение |
|----------|----------|
| Длительность сессии | 12–14 минут |
| Количество активностей | 4–5 |
| Максимум произносительных попыток | 8–10 |
| Смена активности | каждые 3 минуты |
| Пауза внутри сессии | 1 мини-пауза |

**Допустимые шаблоны:** все. Добавляются SC уровень 2, SH, NQ уровень 1–2.

### 8 лет (7 лет 6 месяцев — 8 лет 11 месяцев)

| Параметр | Значение |
|----------|----------|
| Длительность сессии | 13–16 минут |
| Количество активностей | 4–5 |
| Максимум произносительных попыток | 10–12 |
| Смена активности | каждые 3–4 минуты |
| Пауза внутри сессии | 1 мини-пауза (опционально) |

**Допустимые шаблоны:** все без исключений.

### Корректировка при нарушении

**При заикании:** сократить все нормативы на 20–30%, убрать таймеры, не торопить.
**При лёгкой дизартрии:** сократить произносительные попытки на 20%, увеличить ART на 1–2 мин.
**При ОНР:** обычные нормативы, но активности более разнообразные (4 трека параллельно).

## 3. Структура сессии (детально)

### Стандартная сессия (10–12 мин, возраст 6 лет)

| Блок | Длина | Что происходит | Шаблон |
|------|-------|----------------|--------|
| 1. Ритуал приветствия | 15–20 сек | Ляля говорит «привет», называет имя | (анимация) |
| 2. Артикуляционная разминка | 1,5–2 мин | 2–3 упражнения, персонаж показывает | ART или AR |
| 3. Основное упражнение 1 | 2,5–3 мин | Ключевой навык сессии | RAM / L&C |
| 4. Мини-пауза | 20–30 сек | Мини-игра, мини-анимация, «отдыхаем» | (анимация) |
| 5. Основное упражнение 2 | 2–2,5 мин | Смежный или закрепляющий навык | SRT / D&M / MEM |
| 6. Закрепляющая лёгкая игра | 1,5–2 мин | Радостное, знакомое, лёгкое | BNG / MEM |
| 7. Итог + похвала + анонс | 20–30 сек | Ляля хвалит, даёт «задание на дом» (1 фраза) | (анимация) |

Итого: ~10,5 мин.

### Сессия «тяжёлый день» (усталость/болезнь) — 5–7 мин

| Блок | Длина | Что происходит |
|------|-------|----------------|
| 1. Приветствие | 10 сек | Быстро |
| 2. Разминка (1 упражнение) | 1 мин | Самое любимое |
| 3. Лёгкая игра (L&C или MEM) | 3 мин | Только пассивные шаблоны |
| 4. Завершение | 15 сек | Ляля хвалит за то, что пришёл |

### Сессия «хочу ещё» (ребёнок в потоке)

Если ребёнок просит продолжить — добавить одну дополнительную игру (2–3 мин). Не более. Превышение лимита приводит к усталости, которая скажется в следующую сессию.

## 4. Признаки усталости ребёнка

### Поведенческие сигналы (приложение отслеживает косвенно)

**Ранние (лёгкая усталость):**
- Время между нажатием и ответом увеличилось на >2 стандартных отклонения
- Количество попыток до верного ответа выросло без видимой причины
- Ребёнок начал нажимать кнопки быстро, без обдумывания («кликает наугад»)

**Средние (умеренная усталость):**
- 3 подряд ошибки на ранее освоенном материале
- Долгие паузы (>10 сек) без ответа
- Ребёнок нажимает «назад» или «пропустить»

**Поздние (высокая усталость — должны предотвращаться):**
- Ребёнок закрывает приложение
- Ответы случайные (нажимает первое попавшееся)
- Время реакции >15 секунд

### Что отслеживает приложение

```swift
struct FatigueDetector {
  // Порог 1: 3 ошибки подряд
  var consecutiveErrors: Int = 0
  
  // Порог 2: Среднее время ответа выросло >2σ
  var avgResponseTime: TimeInterval = 0
  var responseTimeStdDev: TimeInterval = 0
  
  // Порог 3: Пауза без ответа
  var lastInteractionTime: Date
  let pauseThreshold: TimeInterval = 10
  
  // Порог 4: Кнопка «пропустить» нажата 2+ раз
  var skipCount: Int = 0
  
  func fatigueLevel() -> Int {
    if consecutiveErrors >= 3 || skipCount >= 2 { return 2 }
    if avgResponseTime > (avgResponseTime + 2 * responseTimeStdDev) { return 1 }
    return 0
  }
}
```

## 5. Алгоритм реагирования на усталость

### Уровень 0 (норма)

Сессия продолжается по плану.

### Уровень 1 (лёгкая усталость)

1. Приложение переключает на более лёгкое задание (difficulty downgrade)
2. Следующее задание — пассивный шаблон (L&C вместо RAM)
3. Ляля говорит: «Давай немного отдохнём — посмотрим на красивые картинки?» (30-секундная анимация)
4. После «отдыха» — одно лёгкое задание и финал

### Уровень 2 (умеренная усталость)

1. Ляля говорит: «Ты сегодня хорошо поработал! Может, на сегодня хватит?»
2. Кнопки: «Ещё немного» / «Заканчиваем»
3. При выборе «Ещё немного»: 1 очень лёгкое задание (2 мин) → финал
4. При выборе «Заканчиваем»: стандартный финал с похвалой
5. Родитель получает уведомление: «[Имя] немного устал сегодня, но позанимался [N] минут — это хорошо!»

### Уровень 3 (сильная усталость / отказ)

1. Немедленное завершение сессии
2. Ляля: «Всё, молодец! Завтра продолжим!»
3. Прогресс за текущую сессию сохраняется полностью
4. Не показывать «недовыполненное» — только достигнутое

## 6. Ритуалы начала сессии

### Зачем нужен ритуал

Ритуал (предсказуемая последовательность действий) переводит ребёнка из «обычного» режима в «режим занятий» без стресса. Это уменьшает сопротивление и ускоряет «входное» внимание.

### Стандартный ритуал HappySpeech (30–45 сек)

1. **Заставка** (5 сек): логотип + знакомая мелодия.
2. **Ляля появляется** (5 сек): анимация выхода персонажа.
3. **Приветствие с именем** (5–8 сек): «Привет, [Имя]! Сегодня мы...»
4. **Краткий анонс** (8–10 сек): «Сегодня мы будем тренировать звук [Ш]!» + картинка задания.
5. **Старт** (5 сек): кнопка «Поехали!» или автоматически.

### Вариации ритуала для разных настроений

**Ребёнок активен:**
«Привет! Ляля тоже в хорошем настроении — начинаем скорее!»

**Ребёнок не хочет:**
«Ляля говорит: давай попробуем всего 5 минут. Если захочешь — продолжим, если нет — на сегодня хватит.»

**После долгого перерыва:**
«С возвращением! Ляля соскучилась. Сначала вспомним, что уже умеем.»

### Артикуляционная разминка как часть ритуала

Первые 1,5 минуты всегда — знакомые артикуляционные упражнения. Это не «новая задача», а привычное начало — снижает тревогу входа. Ребёнок знает: «Сначала делаем горку-лошадку — это я умею».

## 7. Ритуалы завершения сессии

### Зачем нужен ритуал завершения

Предсказуемое завершение:
1. Помогает ребёнку «выйти» из игрового режима без обиды
2. Формирует ожидание «я вернусь»
3. Закрепляет позитивное ощущение от занятия

### Стандартный финал (20–30 сек)

1. **Итоговая анимация** (5 сек): Ляля танцует / собирает звёздочки.
2. **Итог** (8–10 сек): «Сегодня ты сделал [N] заданий. Вот твоя звезда!»
3. **Домашнее задание** (5–8 сек): 1 короткая фраза: «Сегодня дома найди 3 слова на [звук]!»
4. **До свидания** (5 сек): «До встречи! Ляля будет ждать!»

### Вариации финала

**Рекордный результат:**
«Сегодня ты справился с [N] заданиями — это твой личный рекорд! Ляля в восторге!»

**После трудной сессии:**
«Сегодня было непросто, но ты не бросил. Это настоящая сила! Завтра — снова вперёд!»

**Если ребёнок хочет ещё:**
«Ляля говорит: до завтра! Отдохни — и завтра продолжим. Так лучше учится язычок.»

## 8. Правила регулярности

### Рекомендуемая частота

| Тип нарушения | Рекомендуемая частота | Минимальная эффективная |
|---------------|----------------------|-------------------------|
| Дислалия (1–2 звука) | 5 раз в неделю | 3 раза в неделю |
| Дислалия (3+ звука) | 5 раз в неделю | 4 раза в неделю |
| ФФН | 5 раз в неделю | 4 раза в неделю |
| ОНР III–IV | 5–7 раз в неделю | 5 раз в неделю |
| ЗРР | 5–7 раз в неделю | 4 раза в неделю |
| Заикание | 5–7 раз в неделю (короткие) | 5 раз в неделю |

### Принцип постепенности

Не начинать с 5 занятий в неделю. Рекомендованный «онбординг»:
- Неделя 1–2: 2–3 занятия, короткие (5–7 мин)
- Неделя 3–4: 3–4 занятия
- С 5-й недели: целевая частота

### Правило непропуска

Пропуск 3+ дней подряд резко снижает эффективность. Это связано с угасанием нейронных связей у незакреплённого навыка. Приложение отправляет напоминание после 2 дней без занятий.

**Формулировка напоминания:** «Ляля скучает! Давно не занимались. Всего 5 минут — и навык сохранится!»

## 9. Перерывы и «каникулы»

### Плановые перерывы

- Перерывы 1–2 дня (выходные) — нормальны, не снижают результат.
- Перерыв 5–7 дней — небольшой регресс (1–2 этапа назад). При возобновлении — начать с закрепляющего повторения.
- Перерыв 2+ недели (болезнь, каникулы) — регресс 1–3 этапа. Возобновлять плавно, с более ранних этапов.

### Сезонные перерывы (лето)

Летом (июнь–август) рекомендовано продолжать занятия, но в облегчённом режиме: 2–3 раза в неделю, короткие сессии (7–8 мин), акцент на удовольствие, а не прогресс.

**Обоснование:** летом активно развивается детский словарный запас через новый опыт. Закрепление звука на новых словах из летнего опыта («краб», «прибой», «ракушка» для Р) — очень эффективно.

### Болезнь

При остром заболевании (температура, боль в горле) — полный перерыв. При выздоровлении — первые 2–3 занятия лёгкие, без произносительных нагрузок.

## 10. Адаптация сессии к состоянию ребёнка

### Определение состояния

Приложение предлагает родителю в начале сессии 3 кнопки:
- «Всё отлично!» → стандартная сессия
- «Немного устал» → укороченная сессия (80% длительности), без RAM уровень 3
- «Плохо себя чувствует» → только лёгкие пассивные задания (5 мин)

### Матрица адаптации

| Состояние | Длительность | Допустимые шаблоны | Произносительные попытки |
|-----------|-------------|-------------------|--------------------------|
| Отлично | 100% | Все | 100% |
| Немного устал | 80% | Без NQ, SC3, RAM3 | 70% |
| Устал | 60% | Только L&C, MEM, BR | 50% |
| Плохо (болен) | 40% | Только L&C уровень 1, анимации | 0 (пассивно) |

### Автоматическое снижение нагрузки без уведомления родителя

Если система обнаруживает признаки усталости по поведению (раздел 4), она тихо снижает нагрузку без прерывания игры:
1. Следующая карточка — более лёгкое слово
2. Дистракторы в L&C — более контрастные (легче выбрать)
3. Темп эталона в RAM — медленнее на 20%
