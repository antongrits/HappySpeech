# Референсный корпус голосов — voice_clone_reference.wav

## Назначение

Этот файл представляет собой синтетический референсный корпус русской речи,
предназначенный для будущей функции клонирования голоса (Voice Cloning, post-v1.0).

Используется как обучающий материал для:
- XTTS-v2 (XTTSv2 Coqui TTS)
- TortoiseTTS
- Любой другой системы клонирования голоса на базе reference audio

## Технические характеристики

| Параметр | Значение |
|---|---|
| Формат | WAV (PCM_16) |
| Частота дискретизации | 16000 Hz (16kHz) |
| Каналы | 1 (mono) |
| Длительность | ~25.9 минут (1553 сек) |
| Размер файла | ~47.4 MB |
| Количество синтетических дикторов | 10 вариантов |

## Структура корпуса

Корпус состоит из 10 последовательных блоков (~155 сек каждый),
разделённых паузами в 0.3 секунды.

## Тексты корпуса

Все 18 текстов посвящены логопедическому контенту для детей 5–8 лет:

1. Свистящие звуки: С, З, Ц (чистоговорки)
2. Шипящие звуки: Ш, Ж, Ч, Щ (чистоговорки)
3. Сонорные звуки: Р, Л (чистоговорки)
4. Заднеязычные: К, Г, Х (чистоговорки)
5. Связный текст — история с ключевыми звуками
6. Скороговорки (Саша на шоссе, Карл у Клары и др.)
7. Детские стихи с проблемными звуками
8. Артикуляционные упражнения — текстовые инструкции
9. Счёт и называние чисел
10. Разнообразные слова по лексическим темам
11–18. Расширенные блоки — история, природа, семья, животные и др.

## Источник

Синтез: Microsoft Azure Neural TTS через edge-tts (публичный браузерный API).

Голоса: `ru-RU-DmitryNeural`, `ru-RU-SvetlanaNeural` — Microsoft Neural Voice.

## Правовой статус

Корпус сгенерирован для академических и исследовательских целей
в рамках дипломного проекта HappySpeech.

Для production voice cloning (post-v1.0) рекомендуется:
- Использовать настоящие детские голоса с письменного согласия родителей
- Соблюдать требования COPPA / GDPR-K / российского законодательства о персональных данных детей
- Не использовать синтетический корпус как замену реальных детских голосов в клинических применениях
