# Что нового в HappySpeech

## v1.0.0 (2026-05-01)

### Главное
- **3D Ляля с эмоциями** — новый маскот RealityKit с 8 состояниями
- **Фонетический анализ речи** — RussianPhonemeClassifier ML модель + G2P словарь
- **Wav2Vec2 на устройстве** — спектрограммный анализ русской детской речи
- **Спектрограммный визуализатор** — увидь свой звук как картинку

- 18 интерактивных игр с Лялей
- Распознавание букв через Apple Pencil или палец (LetterTracing)
- Поиск предметов через камеру (ObjectHunt)
