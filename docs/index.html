HappySpeech

Логопедическое приложение для детей 5–8 лет

О приложении

HappySpeech помогает детям развивать и корректировать речь в игровой форме под руководством маскота Ляли. Приложение создано для домашней практики и работает совместно со специалистами-логопедами.

Платформа: iOS 17+. Язык интерфейса: русский.
