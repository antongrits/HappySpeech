### 1. Логопотам (лидер русского рынка)

**Платформы:** iOS, Android
**Цена:** Freemium. Месячная подписка 249–890 ₽, 3 мес. ~999 ₽, год 1650–2890 ₽, недельная 395–490 ₽
**App Store рейтинг:** 4.6 / 5 (2200 оценок)
**Целевой возраст:** 2–8 лет
**Разработчик:** BUBULEARN CO
**Количество контента:** 5000+ игр (по заявлению сайта)
**URL:** https://apps.apple.com/ru/app/id1513750934

**Ключевые фичи:**
- Нейросеть «с точностью живого логопеда» — виртуальная диагностика речи
- Цифровая диагностика с диаграммами и рекомендациями
- Артикуляционная гимнастика, дыхательные упражнения
- Нейроупражнения для речи и моторики
- Подготовка к школе: алфавит, цифры, чтение
- Эмоциональный интеллект
- Адаптация для детей с аутизмом
- Онлайн-консультации логопедов и нейропсихологов
- Образовательная лицензия, «Золотой медвежонок-2025»
- Занятия от 5 минут, ребёнок занимается самостоятельно

**Оффлайн:** частично (базовые игры), полный функционал требует сети
**Родительский контур:** личный кабинет, отслеживание прогресса
**Специалистский контур:** онлайн-консультации (отдельный продукт), настройки для преподавателей
**AR/камера:** нет
**On-device ML:** нет подтверждения (сервер)
**Русский язык:** да (основной)

**Что хвалят (топ-3):**
1. Ребёнок занимается самостоятельно, без принуждения
2. За 3 месяца поставлены звуки С, З, Ц, за 2 месяца расширился словарный запас
3. Игровой формат, «больше похоже на игру, чем на урок»

**Что критикуют (топ-3):**
1. Частые отмены и переносы занятий без предупреждения (онлайн-сервис)
2. Навязчивые звонки и общение с ботом вместо специалиста
3. Подписка ощущается как развод — агрессивный маркетинг, обещания расходятся с реальностью

**Сильные стороны:** огромный контент, нейросеть для диагностики, узнаваемый бренд
**Слабые стороны:** нет настоящего on-device ML, нет AR, нет специалистского инструментария, offline ограничен, методология не прозрачна

**Разработчик:** Little Bee Speech (лицензированный SLP)
**URL:** https://apps.apple.com/us/app/id1485607474

**Ключевые фичи:**

- Уровни: слово → фраза → предложение → история
- 6 типов активностей
- Запись голоса для самоконтроля
- Отслеживание данных с графиками прогресса
- Специалистский контур: SLP data collection

**AI/ML:** нет автоматической оценки произношения
**AR/камера:** возможность записи видео с ребёнком
**Оффлайн:** да (статические активности)
**Русский язык:** нет
**Специалистский контур:** продвинутый (SLP data tracking, графики)

**Что хвалят:**
1. Систематический подход к фонемам
2. Удобный сбор данных для SLP

**Что критикуют:**
1. Дорого ($59.99 единовременно или подписка)

## Ключевые дифференциаторы HappySpeech

1. **Единственное приложение** с полной русской логопедической методологией (ФФНР, ОНР, все этапы)
2. **On-device ASR** (WhisperKit) — privacy-first без передачи голоса на сервер
3. **Продвинутый AR** (ARKit blendshapes + MediaPipe Face Mesh 478 landmarks + custom TonguePostureClassifier CNN)
4. **LLM-адаптивность** — персонализация через Qwen2.5 on-device, 25+ decision points
5. **Специалистский контур** — единственный на русском рынке с полным SLP toolset
6. **Kids Category compliant** — строгое соответствие Apple guidelines, нет сторонних трекеров
7. **Полностью бесплатно для пользователя** — никаких paywalls, подписок, in-app purchases
8. **Interactive Tutorial** — coach marks + spotlight + auto-tap демо (нет ни у кого)
9. **Motion видео + мультики** — Remotion-generated для milestones и intros
10. **Vision ML tongue tracking** — кастомный CNN обученный на собственных данных
